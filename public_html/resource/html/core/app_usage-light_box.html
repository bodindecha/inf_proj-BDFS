To open : use app.ui.lightbox.open(a,b);
 which contains the first parameter a as the position of the lightbox that can only be "top", "mid" or "btm" (top middle bottom) & the second parameter b as a JS object {} which contains attributes as so
title: "Heading of the lightbox"
allowclose: true|false /* true allows user to force close with an "тип" at the top right corner of the lightbox and false don't */
autoclose: integer|decimal /* the lightbox will automatically close after the inputted second(s) has passed */
html: 'HTML STRING' /* what will be shown in the content part of lightbox */
 Please note that all of these attributes are optional, but it's not a good idea to leave html as a blank attribute, and be careful if you're leaving the allowclose blank|false it may cause the lightbox to not being able to close until the page reloads so maybe you want to add the attribute autoclose for safety

To close : regularly users will be the ones to close it or the timeout set, but if you want it to be programmatically close, then use app.ui.lightbox.close();
