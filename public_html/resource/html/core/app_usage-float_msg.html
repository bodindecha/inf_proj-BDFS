To control : use app.ui.notify(a,b);
 which contains first parameter a as the command for 0 closes the notification and 1 instantiates a new one & the second parameter b depends on the first parameter as so
/* if the first parameter is 0 then the second parameter must be a htmlDOMobject to the message parent */
document.querySelector("div.msg")
/* if the first parameter is 1 then the second parameter must be an array "[]" which contains two child */
/* first one is a number from 0 - 3 which indicates the message priority as { 0: "done (green)", 1: "info (blue)", 2: "warn (yellow)", 3: "error (red)"} */
/* and the second one is the message contained in the new notification (must be a STRING) */
Please note that all notification will automatically close 45 seconds after their instantiation

Structure example : app.ui.notify(1, [1, "You are testing floating message dialog"]);
